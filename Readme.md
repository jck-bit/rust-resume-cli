## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/jck-bit/rust-resume-cli
    ```

2. Navigate to the project directory:

    ```bash
    cd rust-resume-cli
    ```

3. Build the project:

    ```bash
    cargo build
    ```

4. Run the project:

    ```bash
    cargo run
    ```

## Usage

- Use  'up' and 'down' arrow keys to navigate through Experience, Projects, About, Skills
